# Beginning Git

## 1.Git简介
Git: 分布式版本管理系统

## 2.安装Git

### Linux上安装Git
`sudo apt-get install git`

### MacOS上安装Git
* 通过`homebrew`安装
* 通过`Xcode`安装

### Windows上安装Git
Git官网下载安装程序

## 3.设置用户信息
### 全局设置
`$ git config --global user.name "Userame"`
`$ git config --global user.email "email@xxx.com"`
